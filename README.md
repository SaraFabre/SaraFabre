# 📌 Projetos em Destaque

## 🔹 QA API DummyJSON

Projeto prático de testes de API utilizando **Postman**, **Newman** e a API pública **DummyJSON**.

O objetivo do projeto é validar requisições HTTP, análise de respostas, códigos de status, autenticação, payloads JSON e automação de testes de API.

### Principais entregas:

- Testes de requisições GET
- Testes de requisições POST
- Testes de requisições PUT
- Testes de requisições DELETE
- Validação de status code
- Validação de payload JSON
- Criação de collections no Postman
- Execução automatizada com Newman
- Organização de evidências
- Documentação de testes de API
- Estruturação de cenários positivos e negativos

🔗 [Acessar projeto](https://github.com/SaraFabre/qa-api-dummyjson)

## 🔹 QA Portfolio Automation Exercise

Projeto de automação de testes E2E focado em validação de fluxos funcionais utilizando ferramentas modernas de QA.

O projeto simula cenários reais de testes automatizados, garantindo maior confiabilidade, rastreabilidade e qualidade da aplicação.

### Principais entregas:

- Automação de testes E2E
- Estruturação de cenários automatizados
- Validação de fluxos críticos
- Execução de testes funcionais
- Organização do projeto de automação
- Boas práticas de automação
- Geração de evidências de testes
- Versionamento com Git e GitHub
- Estrutura escalável para novos cenários
- Foco em qualidade e manutenção dos testes

🔗 [Acessar projeto](https://github.com/SaraFabre/qa-portfolio-automation-exercise)
